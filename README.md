# KhetMap 🌾🛰️

**See your land from space.**

Free, open-source satellite-powered field analysis for farmers, researchers, students, and GIS professionals.

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173/` in your browser.

## Features

- 🛰️ **Live Satellite Maps** — ESRI satellite basemap with OpenStreetMap overlay
- 🌿 **Crop Health (NDVI)** — Monitor vegetation vigor via NASA GIBS
- 💧 **Water Index (NDWI)** — Detect waterlogged areas and moisture content
- 🪨 **Soil Health (SAVI)** — Soil-adjusted vegetation analysis
- 🗺️ **Draw & Save Fields** — Mark field boundaries, calculate areas
- 📁 **Import/Export** — KML, GeoJSON, CSV support
- 🌤️ **Weather Overlay** — Current conditions & 5-day forecast (with API key)
- 🌐 **Multi-language** — English, తెలుగు, हिंदी
- 📱 **Offline-ready** — All data saved locally on your device

## Tech

Vite + React 19 + TypeScript + Tailwind CSS v4 + Leaflet

## No Sign-up Required

KhetMap works entirely in your browser with no account needed. All field data is stored locally on your device.
